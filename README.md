# ft_transcendence — squelette
- SPA TypeScript (frontend/)
- API Fastify + SQLite (api/)
- Blockchain Hardhat (blockchain/)
- Docker + Traefik (https)

Démarrage rapide (dev local):
  cd api && npm i
  cd ../frontend && npm i
  cd ../blockchain && npm i
  cd ..
  docker compose up -d
Accès: https://app.localhost (front) / https://api.localhost (api)
